Stream Class Hierarchy for Console Input/Output

What is the stream class hierarchy for console input/output?

The stream class hierarchy for console input/output consists of the following classes:

  • ios - provides the basic I/O services and defines the basic format for all input/output operations.
  • istream - provides input operations for reading from input streams.
  • ostream - provides output operations for writing to output streams.
  • iostream - provides both input and output operations.

How can we use the stream class hierarchy for console input/output in a program?

We can use the stream class hierarchy for console input/output in a program as follows:

#include <iostream>
using namespace std;

int main()
{
    int num;
    cout << "Enter a number: ";
    cin >> num;
    cout << "The number you entered is: " << num << endl;
    return 0;
}

Unformatted Input/Output

What is unformatted input/output?

Unformatted input/output refers to the input/output operations that do not involve any formatting.

How can we perform unformatted input/output in C++?

We can perform unformatted input/output in C++ using the following functions:

  • get() - reads a single character from the input stream.
  • put() - writes a single character to the output stream.
  • getline() - reads a line of text from the input stream.
  • write() - writes a block of data to the output stream.

Formatted Input/Output with ios Member Functions

What is formatted input/output with ios member functions?

Formatted input/output with ios member functions refers to the input/output operations that involve formatting.

How can we perform formatted input/output with ios member functions in C++?

We can perform formatted input/output with ios member functions in C++ using the following functions:

  • setw() - sets the field width for the next output operation.
  • setprecision() - sets the precision for floating-point output.
  • setiosflags() - sets the format flags for the output stream.
  • resetiosflags() - resets the format flags for the output stream.

What is formatting with manipulators?

Formatting with manipulators refers to the use of manipulators to control the format of the output.

How can we perform formatting with manipulators in C++?

We can perform formatting with manipulators in C++ using the following manipulators:

  • endl - inserts a newline character into the output stream and flushes the stream.
  • setw() - sets the field width for the next output operation.
  • setprecision() - sets the precision for floating-point output.
  • setiosflags() - sets the format flags for the output stream.

Stream Operator Overloading

What is stream operator overloading?

Stream operator overloading refers to the overloading of the << and >> operators for input/output operations.

How can we overload stream operators in C++?

We can overload stream operators in C++ using the following syntax:

ostream& operator<<(ostream& os, const MyClass& obj)
{
    os << obj.data;
    return os;
}

istream& operator>>(istream& is, MyClass& obj)
{
    is >> obj.data;
    return is;
}

File Input/Output with Streams

What is file input/output with streams?

File input/output with streams refers to the input/output operations that involve reading from and writing to files.

How can we perform file input/output with streams in C++?

We can perform file input/output with streams in C++ using the following classes:

  • ifstream - provides input operations for reading from input files.
  • ofstream - provides output operations for writing to output files.
  • fstream - provides both input and output operations for files.

Opening and Closing Files

How can we open and close files in C++?

We can open and close files in C++ using the following functions:

  • open() - opens a file.
  • close() - closes a file.

For example:

#include <fstream>
using namespace std;

int main()
{
    ofstream myfile;
    myfile.open("example.txt");
    myfile << "Writing this to a file.\\n";
    myfile.close();
    return 0;
}

Read/Write from File

How can we read from and write to a file in C++?

We can read from and write to a file in C++ using the following functions:

  • getline() - reads a line of text from the input stream.
  • write() - writes a block of data to the output stream.

For example:

#include <fstream>
using namespace std;

int main()
{
    ifstream myfile;
    myfile.open("example.txt");
    string line;
    while (getline(myfile, line))
    {
        cout << line << endl;
    }
    myfile.close();
    return 0;
}

File Access Pointers and their Manipulators

What are file access pointers and their manipulators?

File access pointers and their manipulators are used to control the position of the file access pointer in a file.

How can we use file access pointers and their manipulators in C++?

We can use file access pointers and their manipulators in C++ using the following functions:

  • seekg() - moves the input file access pointer to a specific position in the file.
  • seekp() - moves the output file access pointer to a specific position in the file.
  • tellg() - returns the current position of the input file access pointer.
  • tellp() - returns the current position of the output file access pointer.

Testing Errors during File Operations

How can we test for errors during file operations in C++?

We can test for errors during file operations in C++ using the following functions:

  • fail() - returns true if an error occurred during the last input/output operation.
  • bad() - returns true if the stream is in a bad state.
  • eof() - returns true if the end of the file has been reached.
  • good() - returns true if the stream is in a good state.

For example:

#include <fstream>
using namespace std;

int main()
{
    ifstream myfile;
    myfile.open("example.txt");
    if (!myfile.is_open())
    {
        cout << "Error opening file." << endl;
    }
    else
    {
        string line;
        while (getline(myfile, line))
        {
            cout << line << endl;
        }
        if (myfile.bad())
        {
            cout << "Error reading file." << endl;
        }
        else if (myfile.eof())
        {
            cout << "End of file reached." << endl;
        }
        myfile.close();
    }
    return 0;
}
