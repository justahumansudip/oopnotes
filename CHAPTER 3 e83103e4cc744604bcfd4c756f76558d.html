CHAPTER 3

A Simple Class and Object

Q: What is a class in programming?

A: A class is a blueprint or template for creating objects that define a set of attributes (data members) and behaviors (member functions) that the objects of that class possess.

Example:

class Car {
   public:
      string brand;
      string model;
      int year;
};

Car myCar; // Object of class Car

Q: What is an object in programming?

A: An object is an instance of a class that has its own set of attributes and behaviors.

Example:

myCar.brand = "Ford";
myCar.model = "Mustang";
myCar.year = 2021;

Accessing members of class

Q: How can you access the data members of a class?

A: You can access the data members of a class by using the dot (.) operator.

Example:

cout << "Brand: " << myCar.brand << endl;
cout << "Model: " << myCar.model << endl;
cout << "Year: " << myCar.year << endl;

Initialization of class objects: (Constructor, Destructor)

Q: What is a constructor in programming?

A: A constructor is a special member function of a class that is automatically called when an object of that class is created. It is used to initialize the data members of the object.

Example:

class Car {
   public:
      string brand;
      string model;
      int year;

      Car() {  // Default Constructor
         brand = "";
         model = "";
         year = 0;
      }

      Car(string b, string m, int y) {  // Parameterized Constructor
         brand = b;
         model = m;
         year = y;
      }

      Car(const Car &c) {  // Copy Constructor
         brand = c.brand;
         model = c.model;
         year = c.year;
      }

      ~Car() {  // Destructor
         cout << "Object is being deleted" << endl;
      }
};

Car myCar; // Default Constructor
Car myCar2("Ford", "Mustang", 2021); // Parameterized Constructor
Car myCar3 = myCar2; // Copy Constructor

Q: What is a destructor in programming?

A: A destructor is a special member function of a class that is automatically called when an object of that class is destroyed. It is used to free the memory allocated to the object.

Example:

{
   Car myCar4("Honda", "Civic", 2022);
} // Destructor is called here

Default Constructor

Q: What is a default constructor?

A: A default constructor is a constructor that takes no arguments. It is used to create an object with default values.

Example:

Car myCar; // Default Constructor

Parameterized Constructor

Q: What is a parameterized constructor?

A: A parameterized constructor is a constructor that takes one or more arguments. It is used to create an object with user-defined values.

Example:

Car myCar2("Ford", "Mustang", 2021); // Parameterized Constructor

Copy Constructor

Q: What is a copy constructor?

A: A copy constructor is a constructor that creates a new object by copying the values of an existing object. It is used to create a new object with the same values as an existing object.

Example:

Car myCar3 = myCar2; // Copy Constructor

The Default Copy Constructor

Q: What is the default copy constructor?

A: The default copy constructor is a constructor that is automatically generated by the compiler if no copy constructor is defined explicitly. It performs a shallow copy of the object.

Example:

class Person {
   public:
      string name;
      int age;
};

Person person1;
person1.name = "John";
person1.age = 30;

Person person2 = person1; // Default Copy Constructor

Objects as Function Arguments

Q: Can objects be used as function arguments?

A: Yes, objects can be used as function arguments.

Example:

void printCarDetails(Car c) {
   cout << "Brand: " << c.brand << endl;
   cout << "Model: " << c.model << endl;
   cout << "Year: " << c.year << endl;
}

printCarDetails(myCar2); // Object used as function argument

Returning Objects from Functions

Q: Can objects be returned from a function?

A: Yes, objects can be returned from a function.

Example:

Car createCar(string b, string m, int y) {
   Car c(b, m, y);
   return c;
}

Car myCar5 = createCar("Toyota", "Corolla", 2021); // Object returned from function

Structures and Classes

Q: What is the difference between a structure and a class in programming?

A: In C++, a structure is a collection of variables of different data types that are grouped together under a single name. A class is a user-defined data type that contains variables and functions.

Example:

struct Point {
   int x;
   int y;
};

class Rectangle {
   public:
      int length;
      int width;

      int area() {
         return length * width;
      }
};

Memory allocation for Objects

Q: How is memory allocated for objects in programming?

A: Memory is allocated for objects using the new operator. The new operator returns a pointer to the memory location of the object.

Example:

Car *myCar6 = new Car("Nissan", "Sentra", 2022); // Memory allocated for object using new operator

Static members

Q: What are static members in programming?

A: Static members are class members that are shared by all objects of the class. They are initialized only once and remain in memory throughout the lifetime of the program.

Example:

class Counter {
   public:
      static int count;

      Counter() {
         count++;
      }
};

int Counter::count = 0; // Initialization of static member

Counter c1;
Counter c2;
Counter c3;

cout << "Number of objects created: " << Counter::count << endl; // Accessing static member

Member functions defined outside the class

Q: Can member functions be defined outside the class in programming?

A: Yes, member functions can be defined outside the class using the scope resolution operator (::).

Example:

class Rectangle {
   public:
      int length;
      int width;

      int area();
};

int Rectangle::area() {  // Member function defined outside the class
   return length * width;
}
