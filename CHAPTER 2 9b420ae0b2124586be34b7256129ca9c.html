CHAPTER 2

C++ Program Structure

  1. What is the basic structure of a C++ program?
    • The basic structure of a C++ program consists of a header file, a main function, and one or more user-defined functions.
  1. What is the purpose of the header file in a C++ program?
    • The header file contains declarations of functions, variables, and constants that are used in the program.
  1. What is the purpose of the main function in a C++ program?
    • The main function is the entry point of the program and is responsible for controlling the flow of the program.

Character Set and Tokens

  1. What is a character set in C++?
    • A character set is a set of characters that can be used in C++ programs.
  1. What are tokens in C++?
    • Tokens are the smallest units of a C++ program and include keywords, identifiers, operators, and literals.

Data Type

  1. What is a data type in C++?
    • A data type is a classification of data that determines the type of operations that can be performed on the data.
  1. What are the basic data types in C++?
    • The basic data types in C++ include integer, character, boolean, floating-point, and double floating-point.
  1. What is the size of an integer data type in C++?
    • The size of an integer data type in C++ depends on the platform and can be either 2 or 4 bytes.

Type Conversion

  1. What is type conversion in C++?
    • Type conversion is the process of converting a value from one data type to another data type.
  1. What is implicit type conversion in C++?
    • Implicit type conversion is the automatic conversion of a value from one data type to another data type without the need for explicit conversion.
  1. What is explicit type conversion in C++?
    • Explicit type conversion is the manual conversion of a value from one data type to another data type using a type cast operator.

Preprocessor Directives

  1. What are preprocessor directives in C++?
    • Preprocessor directives are statements that are processed by the preprocessor before the compilation of the program.
  1. What is the purpose of the #include directive in C++?
    • The #include directive is used to include the contents of a header file in the program.

Namespace

  1. What is a namespace in C++?
    • A namespace is a mechanism for grouping related declarations and preventing naming conflicts.
  1. How is a namespace declared in C++?
    • A namespace is declared using the namespace keyword followed by the name of the namespace.

Input/Output Streams and Manipulators

  1. What are input/output streams in C++?
    • Input/output streams are used to read input from and write output to external devices such as the console or files.
  1. What are manipulators in C++?
    • Manipulators are special functions that are used to modify the behavior of input/output streams.

Dynamic Memory Allocation with new and delete

  1. What is dynamic memory allocation in C++?
    • Dynamic memory allocation is the process of allocating memory at runtime.
  1. How is dynamic memory allocated in C++?
    • Dynamic memory is allocated using the new operator and deallocated using the delete operator.

Control Statements

  1. What are control statements in C++?
    • Control statements are used to control the flow of a program.
  1. What are the types of control statements in C++?
    • The types of control statements in C++ include if-else, switch, while, do-while, and for.

Functions

  1. What is a function in C++?
    • A function is a block of code that performs a specific task.
  1. What is function overloading in C++?
    • Function overloading is the process of defining multiple functions with the same name but different parameters.
  1. What are inline functions in C++?
    • Inline functions are functions that are expanded by the compiler at the point of function call.
  1. What are default arguments in C++?
    • Default arguments are values that are automatically assigned to function parameters if no argument is provided.
  1. What is pass by reference in C++?
    • Pass by reference is a method of passing arguments to a function by reference instead of by value.
  1. What is return by reference in C++?
    • Return by reference is a method of returning values from a function by reference instead of by value.
  1. What is scope in C++?
    • Scope is the region of the program where a variable can be accessed.
  1. What are storage classes in C++?
    • Storage classes are used to specify the lifetime and scope of variables.

Pointers

  1. What is a pointer in C++?
    • A pointer is a variable that stores the memory address of another variable.
  1. How is a pointer variable declared and initialized in C++?
    • A pointer variable is declared using the * operator and can be initialized with the address of another variable.
  1. What are the operators used with pointers in C++?
    • The operators used with pointers in C++ include the dereference operator (*), the address-of operator (&), and the pointer arithmetic operators (+ and -).
  1. What is the relationship between pointers and arrays in C++?
    • Pointers and arrays are closely related in C++, and arrays can be accessed using pointer arithmetic.
  1. How are pointers used with functions in C++?
    • Pointers are often used with functions in C++ to pass arguments by reference and to return values by reference.
