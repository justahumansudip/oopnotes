Function Templates

  1. What is a function template in C++?

A function template is a blueprint for creating a family of functions with a single definition. It allows you to define a function once and use it with different data types.

Example:

template<typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}
  1. How do you call a function template in C++?

You can call a function template by specifying the data types inside angle brackets when calling the function.

Example:

int x = 5, y = 10;
cout << max<int>(x, y) << endl;

Function Templates with Multiple Arguments

  1. How do you create a function template with multiple arguments in C++?

You can create a function template with multiple arguments by specifying the data types inside angle brackets for each argument.

Example:

template<typename T1, typename T2>
void print(T1 a, T2 b) {
    cout << a << " " << b << endl;
}
  1. How do you call a function template with multiple arguments in C++?

You can call a function template with multiple arguments by specifying the data types inside angle brackets for each argument when calling the function.

Example:

int x = 5;
double y = 10.5;
print<int, double>(x, y);

Class Templates

  1. What is a class template in C++?

A class template is a blueprint for creating a family of classes with a single definition. It allows you to define a class once and use it with different data types.

Example:

template<typename T>
class Stack {
private:
    T* data;
    int size;
public:
    Stack(int max_size) {
        data = new T[max_size];
        size = 0;
    }
    void push(T value) {
        data[size++] = value;
    }
    T pop() {
        return data[--size];
    }
};
  1. How do you create an object of a class template in C++?

You can create an object of a class template by specifying the data type inside angle brackets when creating the object.

Example:

Stack<int> s(10);
s.push(5);
s.push(10);
cout << s.pop() << endl;

Templates and Inheritance

  1. How do you use templates with inheritance in C++?

You can use templates with inheritance by creating a derived class from a base class template and specifying the data type inside angle brackets.

Example:

template<typename T>
class Base {
public:
    virtual void print(T value) = 0;
};

template<typename T>
class Derived : public Base<T> {
public:
    void print(T value) {
        cout << value << endl;
    }
};
  1. How do you create an object of a derived class template in C++?

You can create an object of a derived class template by specifying the data type inside angle brackets when creating the object.

Example:

Derived<int> d;
d.print(5);

Exceptional Handling (Try, Throw and Catch)

  1. What is exceptional handling in C++?

Exceptional handling is a mechanism in C++ that allows you to handle runtime errors in a program. It involves three keywords: try, throw and catch.

  1. How do you use the try keyword in C++?

The try keyword is used to enclose the code that may throw an exception.

Example:

try {
    int x = 10 / 0;
} catch (...) {
    cout << "Error: Division by zero" << endl;
}
  1. How do you use the throw keyword in C++?

The throw keyword is used to throw an exception.

Example:

void divide(int x, int y) {
    if (y == 0) {
        throw "Division by zero";
    }
    cout << x / y << endl;
}
  1. How do you use the catch keyword in C++?

The catch keyword is used to catch an exception and handle it.

Example:

try {
    divide(10, 0);
} catch (const char* msg) {
    cout << "Error: " << msg << endl;
}

Use of Exceptional Handling

  1. Why is exceptional handling important in C++ programming?

Exceptional handling is important in C++ programming because it allows you to handle runtime errors in a program and prevent it from crashing. It also provides a way to gracefully exit a program in case of an error.

  1. What are some common exceptions that can occur in a C++ program?

Some common exceptions that can occur in a C++ program include division by zero, out of memory, null pointer access, file not found, and invalid input.

  1. How can you use exceptional handling to improve the reliability of a C++ program?

You can use exceptional handling to improve the reliability of a C++ program by catching exceptions and displaying informative error messages to the user. This allows the user to understand what went wrong and take appropriate action.
